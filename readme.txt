Ce plugin vous permet d'envoyer des SMS via un compte SMS d'OVH

== Description ==

SMS OVH est un module WordPres pour envoyer des SMS via un compte OVH.

*    Il permet de cr&eacute;er diff&eacute;rentes bases de donn&eacute;es et pour chacune d'y saisir des num&eacute;ros de t&eacute;l&eacute;phone mobile.
 *    Vous pouvez ensuite cr&eacute;er des messages et envoyer le message &agrave; la base de donn&eacute;es de votre choix.
 *    Un historique des envois est disponible.

== Installation ==

Activer le plugin et ensuite indiquer votre login et mot de passe OVH li&eacute; au compte SMS.
